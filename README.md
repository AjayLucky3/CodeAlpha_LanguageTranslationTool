# 🌍 Language Translation Tool

## 📌 Description
A simple AI-powered Language Translation Tool built using Flask and Deep Translator. Users can enter text, select source and target languages, and receive translated text instantly.

## 🚀 Features
- Translate text between multiple languages
- Simple and responsive interface
- Built with Flask
- Uses Google Translator through deep-translator

## 🛠️ Tech Stack
- Python
- Flask
- HTML
- CSS
- JavaScript
- Deep Translator

## 📂 Project Structure
```
CodeAlpha_LanguageTranslationTool/
│
├── app.py
├── requirements.txt
├── templates/
│   └── index.html
├── static/
│   ├── style.css
│   └── script.js
└── README.md
```

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python3 app.py
```

Open:

```
http://127.0.0.1:5000
```
